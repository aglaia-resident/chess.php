# Clear Board

to clear all piece in board, we simply call clear()

## example

```php
<?php
use \Ryanhs\Chess\Chess;

$chess = new Chess();
$chess->clear();
echo 'FEN => ' . $chess->generateFen() . PHP_EOL;
echo $chess;
```
