# Undo :-p

yeah i know, not complete without "undo" right? haha

simply: Ì€ $chess->undo();`

```php
<?php
use \Ryanhs\Chess\Chess;

$chess = new Chess();

// do a move
$chess->move('e4');
echo $chess;

// undo again
$chess->undo();
echo $chess;
```

## return

undo will return NULL if undo failed

```php
<?php
use \Ryanhs\Chess\Chess;

$chess = new Chess();
$chess->undo(); // return NULL
```
